# PeaceFlow App

Một ứng dụng hỗ trợ quản lý căng thẳng và chăm sóc sức khỏe tâm thần cá nhân hóa.

## Tính năng chính

- **Đánh giá mức độ căng thẳng**: Các bài kiểm tra nhanh giúp xác định trạng thái tâm lý.
- **Nhiệm vụ chữa lành**: Danh sách các hoạt động thực hành (thở, thiền, viết nhật ký) được cá nhân hóa.
- **AI Therapist**: Trò chuyện cùng trợ lý ảo hỗ trợ tâm lý.
- **Kết nối chuyên gia**: Tìm kiếm và đặt lịch với các chuyên gia tâm lý uy tín.
- **Cộng đồng**: Nơi chia sẻ và kết nối với những người cùng hành trình.

## Công nghệ sử dụng

- HTML5 / CSS3 (Vanilla)
- JavaScript (ES6+)
- Google Fonts (Inter, Outfit, Playfair Display)
- Lucide Icons

## Cài đặt và sử dụng

Chỉ cần mở file `index.html` trên trình duyệt web của bạn để bắt đầu trải nghiệm.
